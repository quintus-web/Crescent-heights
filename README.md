Crescent Heights School Website
Overview

Crescent Heights is a modern, responsive school website built with HTML and CSS.
It showcases the school’s vision, mission, motto, core values, and provides functional pages for Home, About, Admissions, and Contact.

The website is mobile-friendly, features a sticky navbar, hover effects, and responsive forms and buttons.

Features

Responsive design — works on desktops, tablets, and smartphones

Sticky navigation bar with hover and active effects

Hero section on homepage with call-to-action button

About page structured with Motto → Vision → Mission → Core Values

Admissions page with styled form for applications

Contact page with functional contact form

Card layouts for features and sections

Modern UI/UX with hover, click, and focus effects

Fully mobile-optimized and touch-friendly

How to Run

Clone or download the repository.

Open the project folder.

Open any of the .html files in a web browser (e.g., index.html).

No server is required since this is a static website.

The website is fully responsive and will adapt automatically to different screen sizes.

Customization

Replace images/logo.png with your own school logo.

Update text content in index.html, about.html, admissions.html, and contact.html to match your school information.

Change color scheme by editing the CSS linear-gradient values or primary colors (#0077ff, #00c6ff, #002b5b).
